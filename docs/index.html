Bank Statement Processor

A lightweight Python utility that consolidates multiple Excel bank statement exports into one clean workbook â€” ready for filtering, reconciliation, and analysis.

Quick start

cd run
python -m venv .venv
pip install -r ../requirements.txt
python bank_statement_processor.py

The output file consolidated_statements.xlsx is created in the same folder (run/).

What you get

  • One sheet: Transactions
  • One row per transaction
  • Source_Bank column for traceability (derived from the input filename)
